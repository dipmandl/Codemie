# Release Notes Dashboard (MVP)

A simple dependency-free web dashboard to manage and view release notes.

## Features

- Create a release note
- Display list of releases
- Each release includes:
  - Product name
  - Version
  - Title
  - Description
  - Release date
  - Breaking change flag (yes/no)
- Filter by product name
- Filter by breaking/non-breaking changes
- Data persistence using browser local storage

## Run Locally

1. Open `index.html` in your browser.
2. Start adding release notes from the form.

No build step and no external dependencies are required.
